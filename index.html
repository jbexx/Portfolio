Projects

Sidewinder

Sidewinder aims to bring software into the construction industry at the level of each trade being performed. It is intended to be used in the whole process of re-modeling a multi-family unit housing complex, from due diligence to final construction. It will track the progress of the project at each trade and therefore quantify the progress of the entire project reducing the complexity of oversight.

Sidewinder is software that brings service to the construction industry. It is built in Angular 7 and utilizes Google's material design.

BlackTop

BlackTop's goal is to provide SaaS to the dispatch and fleet management industry. With BlackTop you can track sub-contracted equipment through geolocation and dispatch your fleet alerting drivers of new deliveries with push notifications through Amazon Web Services (AWS).

BlackTop is a cross platform mobile application built in React Native.

DocSwap

The Idea for DocSwap came from my interaction with newly arriving refugee families to the U.S. One of the struggles they face is reading documentation they recieve through mail.

DocSwap is a Mobile App built in React-Native that implements the Camera functionality to allow the user to take a picture of a document. It is then posted to the Google Cloud Vision API where it is parsed and returned as text.

The user can then select a language they wish to translate the document into. Another post is made to the Google Cloud Translation API where it is translated and returned.
